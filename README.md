# Курс по SQL от [Tayviscon IO](https://github.com/tayviscon-io)
[![Official Project](https://img.shields.io/badge/Tayviscon_IO-official-%2363B175)](https://github.com/tayviscon-io/.github/blob/main/profile/README.md#tayviscon-io-github)

**Yet Another SQL Course** - это курс от компании [Tayviscon IO](https://github.com/tayviscon-io)
из серии «[Yet Another Course](https://github.com/tayviscon-io/yet-another-course)»,
демонстрирующий пользу, классификацию и практические приемы работы с SQL, которые вы можете использовать для решения
типовых задач: от написания запросов и оптимизации производительности до управления транзакциями и проектирования
схем данных.

Стоит отметить, что вы можете вполне успешно работать с базами данных, зная лишь основы SQL, однако, освоив эти приёмы,
вы получаете ещё один инструмент в свой личный набор профессионала.
